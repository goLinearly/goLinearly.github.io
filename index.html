수학에 대한 생각의 관점..

우리가 알고있던 수학은 단지 어렵기만하고 점수는 공부한 만큼 나오지 않는 이론적인 수학아니였을까요? 이제는 수학이라는 과목은 우리들의 실생활에서 필요한 순간순간마다 적적하게 도음을 줄 수 있는 친구로 생각을 하게 될것입니다. 코딩을 배우시거나 현재 코딩과 관련된 일을 하고계신분 하지만 수학의 사용도를 잘 모르시는분들께 적극추천!

강의목적..

수학에 대한 우리의 고정관념은 항상 "수학이 어렵다"와 "수학은 쓸 곳이 없다"이였죠? 하지만 자세히 들여다보면 수학은 우리의 일상생활에 필요할 때 또 유용히 쓰일 때가 많습니다. 또한 생각을 해보면 우리에 뇌는 무의식적으로 아주 다양한 상황에서 수학을 응용해서 우리를 동작시킵니다.

선한수학의 강의 목적은 4차 산업의 취지를 기반으로한 수학과 코딩을 결합시켜 우리가 생활에서 필요로하는 것들을 수학을 통해서 컴퓨터화 하는것이 목적입니다. 여러분들은 이제 수학이라는 것을 내가 또 이 세상이 발전하는데에 있어 정말 필요한 도구라고 생각하시고 수학을 진정으로 즐기시면 됩니다.

강의방식..

여러분들께서 접하게 될 이 강의 방식은 아주 간단합니다. 하나의 소재로 기본 개념을 짚은 뒤에 응용을 하여 컴퓨터화 하게 됩니다. 또한 강의를 들으시는 모든 분들의 원활한 소통과 발전을 위해서 커뮤니티 피드백 공간도 각 강의 하단에 붙여 넣을 것입니다. 그곳에서 여러분들이 배운 내용, 또 여러분들만의 관점과 같은 것을을 나누시면 됩니다.

저희 강의에서는 수학을 코딩에 접목시키는 방향성때문에 모든 수학의 내용들과 요소들은 컴퓨터화 시켜서 강의를 진행할 예정입니다.
예를들어 우리는 일반적으로 숫자를 샐 때에 1부터 새지만, 컴퓨터에서는 0부터 카운트를 하게 됩니다. 그래서 컴퓨터상에서는 첫번째에 있는 대상을 지목할 때에 우리는 두번째 대상이라고 지칭합니다.
다른 예로는 x 라는 것을 벡터로 표기할 때에 우리는 보통 선을 하나 x 위에 그려서 표기합니다. 하지만 위와 같은 이유로, 벡터표기를 컴퓨터화 된 벡터로 표기 할 것입니다. x[i] 의 표기입니다.
그래서 결론적으로 우리는 x 라는 벡터에 5번째 요소를 지칭할 때에는 x5 가아니라 x[4]라고 지칭하게 됩니다.

강의시작전 컴퓨터화 수학문자표기 이해를 더 돕고 싶으시다면 여기를 참조에 주세요.

도구사용..

저희가 수학을 컴퓨터화 하기 위서는 컴퓨터와 컴퓨터 언어가 필요합니다. 아마 컴퓨터는 다 있으실거라 생각하고 이제 컴퓨터에서 컴퓨터 언어를 지정하고 그 언어로 코드를 Souce Code Text Editor 에서 입력을 해야합니다. 현재 관련직종에서 사용되고 있는 텍스트에디터는 아주 많지만, 저희가 쓸 에디터는 바로 아주 강력한 Visual Studio Code 입니다. 여기에서는 저희가 사용하게 될 프로그래밍 언어는 Javascript(.js)와 Python(.py) 입니다. 자바스크립트와 파이썬은 현재 가장 많이 사용되고 있는 프로그래밍 언어중 단연 최고의 언어입니다. 상대적으로는 파이썬의 사용범위가 훨씬 높고 문법도 자바스크립드보다는 쉽지만, 두 언어 모두 아주 유용하고 특시 수학을 접목시킨 기계학습, 딥러닝 많은 프로젝트들이 이 두 언어로 만들어지고 있습니다. 아직 프로그래밍을 해본 경험이 없으시거나 익숙하지 않으신 분들도 걱정마시고 강의를 시작하지면 차차 배우게 될 것입니다. 물론 즐거움도 추가입니다.
